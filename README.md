# 🚀 React Profile Card Project

A modern and responsive **Profile Card UI** built using **React.js** and **CSS**.  
This project displays multiple developer profile cards with images, roles, and descriptions.

---

# 📌 Features

✅ Responsive Profile Cards  
✅ Modern UI Design  
✅ React Functional Components  
✅ CSS Styling  
✅ Hover Effects  
✅ Developer Team Section  
✅ Footer Included
